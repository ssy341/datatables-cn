selectRows

起始版本号: {{ page.since }}

改变选择模式为 row

注意：这个需要使用 Select扩展的支持

说明(Description)

Select扩展有三种选择模式，分别是 cell、rows和 columns, 根据当前选择模式 {% include href/api/select.items.api %}，用户得到最终的数据。

如果当前的模式设置为 row，则这个按钮会被激活，参考 {% include href/api/button.active.api %}

示例

Datatables初始化：提供三个按钮来改变选择模式 

$('#myTable').DataTable( {
    buttons: [
        'selectRows',
        'selectColumns',
        'selectCells'
    ]
} );
