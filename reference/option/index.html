选项(Option)

DataTables是一个可高度配置化的类库，可以在生成HTML tables过程中的所有方面实现定制。所有特性可以通过打开、关闭或者定制来满足你对表格所有的要求。 定制要先定义一个options，然后传入$().DataTable()构造函数，通过定制options的内容来实现定制。 - 例如下面的代码 {% include href/option/features.html param="scrollY" %}和 {% include href/option/features.html param="paging" %} 选项用来允许滚动和禁止分页： 

    $('#myTable').DataTable( {
        scrollY: 300,
        paging: false
    } );

有非常多的配置项可以用来定制你的表格，在下面的文档中都有说明。 如果需要从整体上感知DataTables的初始化选项使用，可以阅读 手册中的配置项。
