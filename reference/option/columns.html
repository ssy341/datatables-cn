columns

起始版本号: {{ page.since }}

设置列特殊的初始化属性

说明

在初始化的时候可以使用本参数来定义列，完整的参数参考下面的相关参数
注意，如果是使用 columns来定义列，那么你的th有多少，就要定义多个个（如果你不指定任何选项可以为null）

数据类型

这个选项可以在以下类型:

  • {% include href/array.js %}

示例

禁止第一列参与过滤(此表格有5列，其他列由于不指定选项，则为null) ，下面两段代码是同样的效果 

$('#example').dataTable( {
  "columns": [
    { "searchable": false },
    null,
    null,
    null,
    null
  ]
} );
$('#example').dataTable( {
  "columnDefs": [ {
      "targets": 0,
      "searchable": false
    } ]
} );
